## 🎯 ОСНОВНАЯ ЦЕЛЬ И УСПЕХ

### Миссия
Создать микро-приложение для расчёта **реальной** доходности календарных спредов с учётом всех скрытых расходов, которое поможет трейдерам принимать обоснованные решения.

### Индикаторы успеха
- Пользователи могут быстро сравнить реальную доходность спредов
- Точные расчёты с учётом комиссий, фандинга и проскальзывания
- Интуитивно понятный интерфейс с цветовой индикацией прибыльности

### Качества опыта
- **Точность** - все расчёты максимально приближены к реальности
- **Скорость** - мгновенное отображение и обновление данных
- **Ясность** - результаты понятны с первого взгляда

## 📊 АНАЛИЗ ПРОБЛЕМЫ

### Основная проблема
Существующие калькуляторы календарных спредов показывают **теоретические** данные без учёта:
- Реальных комиссий входа/выхода
- Фандинга и его влияния на доходность  
- Bid-Ask спредов при исполнении
- Скольжения цен при входе

### Контекст использования
Трейдеры используют приложение для:
- Быстрого сканирования рынка на предмет прибыльных спредов
- Сравнения теоретической и реальной доходности
- Расчёта точной прибыли с учётом размера капитала

### Критический путь
1. Пользователь заходит в приложение
2. Устанавливает размер капитала
3. Просматривает список спредов, отсортированных по реальной доходности
4. Анализирует цветовые индикаторы для быстрой оценки
5. При необходимости переходит к детальному анализу

## ⭐ ОСНОВНЫЕ ФУНКЦИИ

### 1. Главная таблица спредов
- **Что делает**: Отображает топ-10 календарных спредов с реальными расчётами
- **Почему важно**: Быстрое сравнение возможностей без лишней информации
- **Критерий успеха**: Пользователь может за 10 секунд найти лучший спред

### 2. Цветовая индикация доходности
- **Что делает**: Визуально выделяет прибыльные (зелёный), средние (жёлтый) и убыточные (красный) спреды
- **Почему важно**: Мгновенная оценка без анализа чисел
- **Критерий успеха**: 95% пользователей понимают систему без объяснений

### 3. Настройка капитала
- **Что делает**: Пересчитывает прибыль в долларах исходя из размера портфеля
- **Почему важно**: Персонализация расчётов под конкретного трейдера
- **Критерий успеха**: Прибыль отображается корректно для любой суммы капитала

## 🎨 НАПРАВЛЕНИЕ ДИЗАЙНА

### Визуальный тон и идентичность
- **Эмоциональный отклик**: Профессиональность, надёжность, скорость принятия решений
- **Личность дизайна**: Серьёзный финансовый инструмент с элементами современности
- **Визуальные метафоры**: Биржевые терминалы, финансовые дашборды
- **Спектр простоты**: Минимальный интерфейс с фокусом на данных

### Цветовая стратегия
- **Тип схемы**: Тёмная тема с акцентными цветами
- **Основной цвет**: Синий (#3b82f6) - доверие и стабильность
- **Вторичные цвета**: Тёмно-синий фон для профессионального вида
- **Акцентный цвет**: Зелёный для прибыльности, красный для убытков, жёлтый для предупреждений
- **Психология цвета**: Тёмная тема снижает усталость глаз при длительной работе
- **Контрастность**: WCAG AA соответствие для всех текстовых элементов

### Типографическая система
- **Стратегия сочетания**: Roboto для заголовков, IBM Plex Mono для чисел
- **Иерархия**: Крупные заголовки, средние подзаголовки, моноширинные данные
- **Личность шрифта**: Технический, точный, легко читаемый
- **Фокус на читаемости**: Увеличенный межстрочный интервал для таблиц
- **Выбранные шрифты**: Roboto (основной), IBM Plex Mono (данные)
- **Проверка читаемости**: Отлично подходят для финансовых данных

### Визуальная иерархия и макет
- **Направление внимания**: Сначала на реальную доходность, затем на прибыль
- **Философия пустого пространства**: Щедрые отступы для снижения когнитивной нагрузки
- **Сетка**: 8-колоночная сетка для таблицы данных
- **Адаптивный подход**: Mobile-first с плавным переходом к десктопу
- **Плотность контента**: Высокая информативность без загромождения

### Анимации
- **Целенаправленное значение**: Плавные переходы при hover для обратной связи
- **Иерархия движения**: Тонкие анимации только для интерактивных элементов
- **Контекстуальная уместность**: Минимальные анимации для сохранения профессионального вида

### UI элементы и выбор компонентов
- **Использование компонентов**: Card для основной таблицы, Button для действий
- **Кастомизация**: Тёмная тема с синими акцентами
- **Состояния**: Hover эффекты для строк таблицы
- **Выбор иконок**: Phosphor Icons для современного вида
- **Иерархия компонентов**: Первичные кнопки для основных действий
- **Система отступов**: Tailwind spacing scale (4, 6, 8, 12)
- **Мобильная адаптация**: Горизонтальная прокрутка таблицы на узких экранах

### Фреймворк визуальной согласованности
- **Подход системы дизайна**: Компонентный дизайн с переиспользуемыми элементами
- **Ритм**: Последовательные паттерны в размерах и отступах
- **Соответствие бренду**: Финансовая надёжность через визуальную стабильность

### Доступность и читаемость
- **Цель контраста**: WCAG AA соответствие для всех элементов
- Тёмный фон с светлым текстом обеспечивает отличную контрастность

## 🚧 КРАЙНИЕ СЛУЧАИ И ПРОБЛЕМНЫЕ СЦЕНАРИИ

### Потенциальные препятствия
- Отсутствие интернет-соединения (решение: кеширование последних данных)
- Неточные данные API (решение: мок-данные для демонстрации)
- Различные размеры экранов (решение: адаптивный дизайн)

### Обработка крайних случаев
- Нулевая или отрицательная доходность (отображение красным цветом)
- Очень большие числа (форматирование с разделителями тысяч)
- Загрузка данных (скелетон-загрузчик)

## 💭 РЕФЛЕКСИЯ

### Уникальность подхода
Фокус на **реальной**, а не теоретической доходности делает приложение практически полезным для трейдеров, выделяя его среди калькуляторов-конкурентов.

### Проверяемые предположения
- Пользователи предпочитают тёмную тему для финансовых приложений
- Цветовая кодировка интуитивно понятна без легенды
- Моноширинный шрифт улучшает читаемость числовых данных

### Исключительность решения
Сочетание профессионального финансового инструмента с современным, чистым дизайном и мгновенной визуальной обратной связью создаёт превосходный пользовательский опыт для трейдеров.
